# Hot Monitor — 需求文档

> **版本**: v1.2 | **日期**: 2026-06-25

---

## 1. 项目背景

AI 编程博主需要第一时间获取热点动态（如 AI 大模型更新），不能依赖人工手动搜索。需要一个工具自动发现指定热点变化，及时通知。

定位：轻量级工具类项目，敏捷开发，不过度工程化。

---

## 2. 用户角色

- **AI 编程博主**：添加监控关键词，查看实时热点，第一时间获取通知
- **系统**：定时自动扫描、AI 验证真伪、存储并推送

---

## 3. 功能需求

### F1: 关键词监控

| 功能 | 说明 |
|------|------|
| 添加关键词 | 输入关键词 + 可选监控范围（如 "GPT-5"、"AI 编程"） |
| 滑块开关 | 绿色滑块激活/暂停监控 |
| 编辑 | 修改关键词名称和范围 |
| 删除 | 确认后删除 |

### F2: AI 真伪识别

| 功能 | 说明 |
|------|------|
| 内容验证 | 判断搜索结果是否真正与关键词相关 |
| 假冒识别 | 排除同名不同义、蹭热度、标题党 |
| 相关性评分 | 0-100 分 |
| 摘要生成 | AI 生成一句话中文摘要 |

### F3: 多源搜索

7 大免费数据源，无需 API Key：

| 来源 | 类型 | 说明 |
|------|------|------|
| Bing | 国际搜索 | 网页爬虫 |
| Google | 国际搜索 | 网页爬虫 |
| DuckDuckGo | 隐私搜索 | Lite HTML |
| 搜狗 | 中文搜索 | 网页爬虫 |
| HackerNews | 科技社区 | Algolia API |
| B站 | 视频平台 | 官方 API |
| 微博热搜 | 实时热搜 | Ajax API |

> `searchAll` 自动聚合去重。

### F4: 定时扫描

| 功能 | 说明 |
|------|------|
| 自动扫描 | 每 30 分钟自动扫描所有活跃关键词 |
| 手动扫描 | Header 按钮触发全量扫描 |
| 首次扫描 | 启动后 10 秒自动执行 |

### F5: 通知

| 功能 | 说明 |
|------|------|
| 页面轮询 | 15 秒间隔刷新热点流 |
| 新热点计数 | Header 红色 badge |
| 邮件通知 | SMTP 配置后自动发送（待配置） |

### F6: Web 管理界面

| 页面/区域 | 功能 |
|-----------|------|
| 热点流（首页） | 实时展示已验证热点，环形评分指示器 |
| 监控词 | 关键词管理（添加/编辑/删除/开关） |
| 搜索 | 搜索历史热点（标题/摘要/关键词全文检索） |
| 统计卡片 | 关键词数、热点数、24h、AI 验证数 |
| 系统日志 | Header 按钮 → 点击弹窗查看 |
| 设置 | 扫描间隔、邮件 SMTP 配置 |

### F7: Agent Skills（Phase 2）

封装为 Agent Skills，供其他 AI Agent 调用。

---

## 4. 非功能需求

| 维度 | 要求 |
|------|------|
| 部署 | 单命令启动，零外部依赖 |
| 存储 | 本地 SQLite，无需安装数据库服务 |
| 性能 | 前端响应 < 200ms，搜索+AI < 30s |
| 安全 | API Key 通过 .env 管理，.gitignore 排除 |
| 可扩展 | 搜索源可插拔，通知渠道可扩展 |
| UI | 深空玻璃风格，Aceternity 动效，响应式 |
